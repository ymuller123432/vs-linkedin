# VS LinkedIn Website

Static professional site for LinkedIn advertising & marketing services + invoice requests.

## Quick start
Open `index.html` in a browser to preview locally.

## Key files
- `index.html` — The complete site (Tailwind via CDN, self-contained)
- `bimi/logo.svg` — Your BIMI-compliant logo (this will be served at https://vs-linkedin.com/bimi/logo.svg after deployment)

## Deployment
See `DEPLOY.md` for Cloudflare Pages instructions (recommended).

Once deployed to the root domain you can use the logo URL directly in your BIMI record.

## Form
Currently demo mode (submissions log to console + nice success UI). 
Easy to connect to Formspree, Web3Forms, or a Worker for real email delivery.

## Brand note
Update any placeholder text, contact details, or stats as needed. The design uses LinkedIn blue (#0A66C2) for familiarity.
